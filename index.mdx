BedrockConnect is a free, open-source server that gives Minecraft: Bedrock Edition players on game consoles access to a custom server list. By intercepting traffic for featured servers via DNS, it presents an in-game UI where players can add, remove, and join any Bedrock server — including servers that consoles wouldn't otherwise support.

## What BedrockConnect does

Console versions of Minecraft: Bedrock Edition restrict the in-game server browser to a small list of Microsoft-approved "featured servers." BedrockConnect works around this limitation — without modifying your console — by redirecting DNS queries for those featured server domains to a BedrockConnect instance instead. Once connected, you see a server-list interface where you can enter any IP and port to join.

## Quick start

<Steps>
  <Step title="Change your console's DNS">
    Set your primary DNS to a BedrockConnect instance. For Xbox and Switch, use `104.238.130.180`. For PlayStation, use `45.55.68.52`.
  </Step>
  <Step title="Open Minecraft and go to Servers">
    Navigate to the **Servers** tab in Minecraft's main menu.
  </Step>
  <Step title="Join a redirect-compatible featured server">
    Select one of the compatible featured servers: **Mineville**, **Lifeboat**, **Enchanted**, **Galaxite**, or **The Hive**. This opens the BedrockConnect server list.
  </Step>
  <Step title="Enter your server IP and connect">
    Use the BedrockConnect UI to add your server's IP and port, then connect.
  </Step>
</Steps>

<Note>
  BedrockConnect is and always has been free and open-source since 2019. There is no official mobile app — any app on the Apple or Google Play store using this name is not affiliated with this project.
</Note>
