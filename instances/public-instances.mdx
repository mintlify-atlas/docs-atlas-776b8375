Several community members host public BedrockConnect instances you can use without running your own server. Each instance runs independently, so you can connect to any of them using the DNS method or direct IP depending on what the instance supports.

<Note>
  Each instance maintains its own separate player database. If you save servers on one instance and later connect to a different one, you will need to add your servers again.
</Note>

## DNS method vs. direct connection

Instances marked **Yes** for DNS method support the recommended setup: you change your console's DNS settings to the instance IP, then connect through a featured server in-game. This works on Nintendo Switch, Xbox, and PlayStation without any additional software.

Instances marked **No** do not run a DNS server. You can still reach them, but you'll need an alternative join method such as the [Add Friend method](https://github.com/MCXboxBroadcast/Broadcaster?tab=readme-ov-file#standalone) or a LAN connection if your console is on the same network.

## PlayStation note

The main instance at `104.238.130.180` works for Switch and Xbox. For **PS4 and PS5**, use `45.55.68.52` as your primary DNS address instead. This is a dedicated entry point for the same main instance optimized for PlayStation.
