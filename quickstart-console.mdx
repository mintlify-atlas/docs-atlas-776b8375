BedrockConnect gives you access to any Minecraft: Bedrock Edition server from your game console in just a few minutes. The fastest way to get connected is the DNS method — no downloads or account changes required. Alternatively, the Add Friend method lets you join through the Friends menu using a separate companion tool.

## DNS method

The DNS method redirects traffic for featured server domains to a BedrockConnect instance. When you join a compatible featured server, you land on the BedrockConnect server-list UI instead of the actual featured server. From there you can enter any IP and port to transfer to your destination.

<Tabs>

        | Primary DNS | `104.238.130.180` |

## Add Friend method

The Add Friend method uses the **Join Game** button in Minecraft's Friends menu to redirect you to a BedrockConnect instance. It relies on [MCXboxBroadcast](https://github.com/MCXboxBroadcast/Broadcaster) — a separate tool that advertises a BedrockConnect server as an Xbox Live friend session so you can join through the Friends tab.

<Warning>
  The public BedrockConnect bots for the Add Friend method are currently offline due to scalability and security concerns. To use this method, you need to run your own MCXboxBroadcast instance pointing at a BedrockConnect server.
</Warning>

To set this up yourself, follow the [Standalone install instructions in the MCXboxBroadcast repo](https://github.com/MCXboxBroadcast/Broadcaster?tab=readme-ov-file#standalone) and set the target server address in `config.yml` to your BedrockConnect instance IP (for example, `104.238.130.180` for the main public instance).
